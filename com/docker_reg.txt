Lab 9: Container Registry and Image Management 
1. Build a Docker image from a simple app.
 2. Tag the image and push it to Docker Hub or a private registry. 
3. Pull the same image on another machine and run it. 
4. Scan the image for vulnerabilities and save the report.














































































STEP 1 — Build a Docker Image From a Simple App




app/
  app.py
  Dockerfile






app.py




from flask import Flask
app = Flask(__name__)


@app.route('/')
def hello():
    return "Hello from my Docker App!"






Dockerfile


FROM python:3.10
WORKDIR /app
COPY . .
RUN pip install flask
CMD ["python", "app.py"]




docker build -t myapp:1.0 .






STEP 2 — Tag the Image and Push to Registry






docker login


docker tag myapp:1.0 username/myapp:1.0






username/imagename:version


docker push username/myapp:1.0






STEP 3 — Pull Image on Another Machine


docker pull username/myapp:1.0
docker run -p 5000:5000 username/myapp:1.0






STEP 4 — Scan the Image for Vulnerabilities


docker scout quickview


docker scout cves myapp:1.0


docker scout cves myapp:1.0 --output cves-report.json
