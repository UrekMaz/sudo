2. Create or update a security group to allow SSH.

Install Web Server on EC2
Once logged in, run:
sudo yum update -y
sudo yum install httpd -y
sudo systemctl start httpd
sudo systemctl enable httpd


Create the webpage (use single quotes!)
echo '<h1>Welcome to AWS Lab 5 - EC2 Web Server!</h1>' | sudo tee /var/www/html/in




Allow HTTP (Port 80) in Security Group
                                       1. Go to EC2 Dashboard → Instances

                                       2. Select your instance

                                       3. Click Security tab

                                       4. Click your Security Group ID

                                       5. Click Edit inbound rules

                                       6. Add:

Type
	Port
	Source
	HTTP
	80
	0.0.0.0/0
